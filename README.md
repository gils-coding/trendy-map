# 🗺️ 트렌디맵 (Trendy-Map)

> 지금 유행 중인 음식 매장을 지도에서 바로 찾아보세요.

버터떡, 두바이 쫀득쿠키, 소금빵, 탕후루, 크로플, 마라탕 — 최신 트렌드 음식 매장을 내 주변에서 손쉽게 탐색할 수 있는 지도 서비스입니다.

**→ 바로 사용하기: [trendmap.up.railway.app](https://trendmap.up.railway.app)**

---

## 주요 기능

- **카테고리 탭** — 버터떡 / 두바이 쫀득쿠키 / 소금빵 / 탕후루 / 크로플 / 마라탕
- **내 주변 검색** — 현재 위치 기준으로 주변 매장 자동 표시
- **지역 검색** — "홍대", "강남" 등 지역명 또는 매장명으로 검색
- **매장 정보** — 주소, 전화번호, 영업시간, 카카오맵 / 네이버지도 바로가기
- **매장 제안** — 누락된 매장을 직접 제보할 수 있는 기능

---

## 사용 방법

### 1. 카테고리 선택
상단 탭에서 원하는 음식 카테고리를 클릭하면 지도에 주변 매장이 표시됩니다.

### 2. 지역 검색
검색창에 지역명이나 매장명을 입력하세요.
예: `마포구`, `홍대 버터떡`, `이소양과점`

### 3. 내 위치로 이동
위치 버튼을 누르면 현재 위치 주변 매장을 바로 탐색할 수 있습니다.

### 4. 매장 정보 확인
마커 또는 목록에서 매장을 클릭하면 상세 정보와 외부 지도 링크가 나타납니다.

### 5. 매장 제보
지도에 없는 매장을 발견하셨나요? 하단의 **매장 제안** 버튼으로 제보해주세요.
